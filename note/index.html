1.springboot项目打包成docker镜像

1.1 完成springboot项目，打包成jar包

1.2 制作Dockerfile文件，写入脚本

1.3 把jar包以及Dockerfile放到服务器上（已经安装好Linux环境）

在目录下执行命令即可：
docker build -t uploaddemo .

1.4 发现docker已经有了这个镜像

1.5 但是不想把文件保存到容器中，而是容器外，需要目录映射
